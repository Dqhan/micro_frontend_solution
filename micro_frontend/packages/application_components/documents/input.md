### 使用方式
```typescript
import InputComponent from 'component_library/Input';

const Demo = () => {
  return (
    <div>
      <InputComponent />
    </div>
  );
};
```

### API
| 参数 | **说明** | **类型** | **默认值** |
| --- | --- | --- | --- |
| **value** | 输入框内容 | string | - |
| **onChange** | 输入框内容变化时的回调 | function(e) | - |
